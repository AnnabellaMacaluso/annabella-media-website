How to Animate in Photoshop

By Annabella Macaluso 10 April 2020, Friday

Creating a website is a childhood fantasy I've always had, but never achieved until now. Being a cheapskate, I didn't want to accrue the cash needed to make a website, so I managed to find a way to do it for free. Currently this method only applies to individuals who are in school, but the process is still widely applicable. Nonetheless let's dive in!

1. Acquiring a Domain

Github has a lowkey educational site where it provides free access to computer programs and tools if you're a teacher or student. One of these incentives is a free domain provided by name.com.

Visit education.github.com and apply for their student/teacher pack. This pack requires a school id card for identification if you're a student. I also recommend using a school email address as it makes the application process faster and you have a higher chance of getting accepted. I used my personal email address at first and I was rejected :( but, my school email was accepted.

Once your application is accepted, sign up for a domain through name.com.

2. Hosting the Website

Woohoo! Now you have your domain. But, there's no way for people to see it just yet. That's why you need a web hosting service. The one I use is by Googiehost. All you need to do is apply for their free hosting service and use the "I will use my existing domain and update my nameservers option". Insert the domain name you just created.

After you've signed up, you need to change the nameservers on name.com. Changing the nameservers allows Googiehost to manage your DNS and it will update any changes you make. But, the domain will still be registered under name.com.

Now you have to wait for at least 2 days for these changes to take place and for the website to propagate. Googiehost will email you a temporary static IP you can visit to begin making changes to your website but DO NOT share this link. IP addresses should be kept a secret.

3. Coding the Website

Now the fun part begins! In my opinion. You can code your website in any way you'd like. I recommend searching for free templates online to use as it'll make the process easier than starting from scratch. Also since you have the Github package you get many of JetBrains programming IDEs for free. I suggest checking out PHPStorm to get started with programming your website.

Once you've completed programming your website, Googiehost will email your Control Panel URL and login. Here you can upload your code in the File Manager to the public_html folder.

Conclusion

Overall, I hope you found this helpful! Now you can begin setting up whatever sort of website your heart desires. ;-)
